Lưu ý khi chạy project

1 Folder Model cần được để vào thư mục htdoc của XAMPP
2 Mở Command Prompt(cmd) trong thư mục chứa code và gõ 'npm install' và 'npm start' để chạy api
3 Cần nhập file sql vào phpmyadmin 
4 Lưu ý thay đổi các cổng localhost phù hợp trong các file javascript để chạy project


Các Code convention mà project áp dụng
Quy tắc đặt tên: 
- Thể hiện được mục đích
- Càng ngắn càng tốt (ít hơn 20 ký tự)
- Tránh các tên tương tự nhau

Quy tắc xuống hàng: 
- Nếu có nhiều cặp lồng ghép với nhau thì cần xuống hàng theo từng cấp.
- Nếu có dấu "," thì xuống hàng sau dấu ",".
Quy tắc số lượng: Một dòng Code không nên dài quá 80 ký tự

Tabs indentation: với indentation level sử dụng 1 tab
